# Double J Gulf Services — Full Stack Logistics Platform

A complete logistics platform built for Gulf operations, including:

- Customer portal
- Admin dashboard
- Multi-step order wizard
- Saved delivery locations
- Customer profile management
- Order filters + timeline
- SEO-optimized public website
- Full backend API (Node + Express + MongoDB)

## Tech Stack

### Frontend
- React + TypeScript
- Vite
- TailwindCSS
- React Router
- react-helmet-async (SEO)

### Backend
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- CORS
- dotenv

## Features

### Customer Portal
- Login / Register
- Create orders (multi-step wizard)
- Saved delivery locations
- Profile management
- Order history with filters
- Order status timeline

### Admin Dashboard
- Login
- View all orders
- Update order status

### Public Website
- Home
- Services
- About
- Contact
- FAQ
- SEO metadata
- Sitemap + robots.txt
